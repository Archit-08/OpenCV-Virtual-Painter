# 🎨 OpenCV Virtual Painter

A gesture-controlled virtual painter built using **Python**, **OpenCV**, and **MediaPipe**.

---

## ✨ Features

- 🖐️ Real-time hand tracking
- ✏️ Freehand drawing
- 🧽 Gesture-based eraser
- 🖥️ Full-screen interface
- 🎯 Smooth drawing using a persistent canvas

---

## 🛠️ Tech Stack

- Python
- OpenCV
- MediaPipe
- NumPy

## 🚀 Installation

Clone the repository

```bash
git clone https://github.com/Archit-08/OpenCV-Virtual-Painter.git
```

Go into the project folder

```bash
cd "OpenCV Virtual Painter"
```

Install dependencies

```bash
pip install -r requirements.txt
```

Run the project

```bash
python Opencv virtual painter.py
```
